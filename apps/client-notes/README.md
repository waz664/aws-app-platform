# Client Notes

`Client Notes` is a sample app concept for validating the baseline architecture.

## Purpose

An authenticated app for tracking clients, notes, follow-ups, and lightweight tasks.

## Users

- Owner or admin
- Team member

## Screens

- Sign in
- Dashboard
- Clients list
- Client detail
- Follow-up task board
- Profile and settings

## Data Model

- User
- Client
- Note
- Task

## Why This Is A Good First App

It proves the common platform pieces we expect to reuse across many future apps:

- auth
- responsive dashboard UI
- list and detail pages
- CRUD API patterns
- lightweight data storage

It is small enough to build quickly but realistic enough to pressure-test the repeatable process.
